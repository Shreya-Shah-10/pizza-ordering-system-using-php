# pizza Ordering system using PHP

* How to run this Project

1. Move Project to XAMPP
- Copy the entire project folder.
- Paste it inside:
  C:\xampp\htdocs\
   or
  C:\xampp\htdocs\pizza-ordering-system
  
2. Create the Database
- Start XAMPP (Apache + MySQL).
- Open phpMyAdmin in your browser:
    http://localhost/phpmyadmin
- Click New → Create a database named:
    pizzadb

3. Import the Database
- Inside phpMyAdmin, open database pizzadb.
- Click Import.
- Select the SQL file located in the project’s /database/ folder.
- Click Go to import.

4. Run the Project
- User Panel:
  http://localhost/pizza-ordering-system
